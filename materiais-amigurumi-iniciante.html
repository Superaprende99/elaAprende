🧶 Amigurumi

Materiais para Amigurumi Iniciante — Lista Completa

📅 Atualizado em 2025 · ⏱ Leitura: 5 minutos

Uma das primeiras dúvidas de quem quer aprender amigurumi é: o que preciso comprar? Aqui está a lista completa com tudo que você precisa, quanto custa e onde encontrar.

Materiais essenciais

Agulha de crochê

Tamanho 2,5mm a 3mm é o ideal para amigurumi. Prefira cabo ergonômico.

R$ 8–25

Linha de algodão colorida

Linha específica para amigurumi — mais encorpada e fácil de trabalhar.

R$ 12–25/novelo

Fibra siliconada para enchimento

Deixa os bichinhos macios e bem formados. Compre em pacotes maiores.

R$ 15–30

Olhinhos de segurança

Em vários tamanhos — 6mm, 9mm, 12mm. Dão vida aos personagens.

R$ 10–20

Agulha de lã para costura

Para costurar as partes do bichinho e fazer acabamentos.

R$ 5–10

Tesoura pequena

Para cortar as linhas. Uma boa tesoura de bordado é ideal.

R$ 10–20
Investimento total para começar R$ 60 – R$ 130
💡 Onde comprar mais barato

Shopee e Mercado Livre têm ótimos preços para agulhas e olhinhos. Para linhas, lojas de artesanato locais ou atacados online costumam ter boas opções. Compare preços antes de comprar.

Materiais opcionais para evoluir

  • Marcadores de ponto — para não perder a contagem das carreiras
  • Alfinetes — para fixar as partes antes de costurar
  • Agulha de feltragem — para detalhes em feltro
  • Tinta acrílica — para pintar detalhes nas peças
  • Fio de arame — para peças articuladas

Dicas para escolher a linha certa

  • Para iniciantes: linha de algodão 100% é a mais fácil de trabalhar
  • Evite linhas muito finas no começo — é mais difícil de contar os pontos
  • Linhas anti-pilling são melhores para peças que serão lavadas
  • Para bichinhos pequenos, use linha mais fina (0,5mm a 1mm)
